#### Task 2: Revenue dashboard + Revenue API

**Type:** auto

**Files:**
- `src/app/api/revenue/route.ts`
- `src/app/(admin)/revenue/page.tsx`
- `src/components/financials/revenue-chart.tsx`

**Action:**

1. **Create `src/app/api/revenue/route.ts`:**
   - `GET`: Returns revenue summary. Accept optional query param `months` (default 12) for time series range.
   - Queries:
     - Total Revenue (sum of totalPence where status="paid")
     - Outstanding (sum of totalPence where status in ["sent", "overdue"])
     - Overdue (sum of totalPence where status="overdue")
     - MRR: Average of paid invoice totals in the most recent 3 months (or sum of all active workspace billingRetainerPence + billingPlatformFeePence)
     - Monthly time series: Group paid invoices by month (YYYY-MM), sum totalPence. Return array of `{ month: string, revenuePence: number }` for the last N months.
     - Per-client breakdown: Group paid invoices by workspaceSlug, sum totalPence. Join with workspace name. Return array of `{ workspaceSlug: string, workspaceName: string, totalPaidPence: number, invoiceCount: number }`.
   - Return JSON with all computed data

2. **Create `src/components/financials/revenue-chart.tsx`:**
   - "use client" component
   - Follow the exact same pattern as `src/components/dashboard/activity-chart.tsx` (Recharts AreaChart)
   - Props: `data: { month: string; revenuePence: number }[]`
   - Single area for revenue, use brand color oklch variant or zinc-based color
   - X-axis: month labels (Jan, Feb, etc.)
   - Y-axis: format as GBP using `formatGBP` or `£${value.toLocaleString()}`
   - Custom tooltip matching existing dashboard tooltip pattern

3. **Create `src/app/(admin)/revenue/page.tsx`:**
   - Server component with `export const dynamic = "force-dynamic"`
   - Fetch revenue data from `/api/revenue` (or call prisma directly — server component)
   - Layout:
     - Header: "Revenue" (using `Header` component)
     - **4 KPI metric cards** (use project's MetricCard component or Card component):
       - Total Revenue (paid) — formatGBP
       - Outstanding (unpaid) — formatGBP
       - Monthly Recurring Revenue — formatGBP
       - Overdue — formatGBP, use red/destructive styling if > 0
     - **Revenue chart**: `RevenueChart` component with monthly data (last 12 months)
     - **Per-client breakdown table**: Table with columns: Client, Total Paid, Invoice Count, Average Invoice. Use project's Table components.

**Verify:**
```bash
npx tsc --noEmit 2>&1 | grep -E "revenue" | head -10
```

**Done:** GET /api/revenue returns KPIs + time series + per-client data. /revenue page shows 4 KPI cards, monthly revenue chart, and per-client breakdown table.
