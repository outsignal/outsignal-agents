### Tasks

#### Task 1: Auto-generation logic + overdue detection

**Type:** auto

**Files:**
- `src/lib/invoices/generator.ts`
- `src/lib/invoices/overdue.ts`

**Action:**

1. **Create `src/lib/invoices/generator.ts`:**

   `generateDueInvoices(): Promise<{ created: number; skipped: number }>`:
   - Get current date (today)
   - Calculate windowEnd = today + 7 days
   - Find all workspaces where `billingRenewalDate` is NOT null AND `billingRenewalDate >= today AND <= windowEnd`
   - Also require workspace to have `billingRetainerPence` or `billingPlatformFeePence` set (no point generating a £0 invoice)
   - For each workspace:
     - **Idempotency check:** Look for existing invoice with `{ workspaceSlug: ws.slug, renewalDate: ws.billingRenewalDate, status: { in: ["draft", "sent"] } }`. If found, skip.
     - Calculate `billingPeriodEnd` = ws.billingRenewalDate
     - Calculate `billingPeriodStart` = billingPeriodEnd minus 1 month (use `advanceRenewalDate` inverse — subtract 1 month)
     - Build line items array from workspace billing config:
       - If `billingRetainerPence` > 0: `{ description: "Cold Outbound Retainer (${formatPeriod(start)} - ${formatPeriod(end)})", quantity: 1, unitPricePence: ws.billingRetainerPence }`
       - If `billingPlatformFeePence` > 0: `{ description: "Cold Outbound Platform Fee (${formatPeriod(start)} - ${formatPeriod(end)})", quantity: 1, unitPricePence: ws.billingPlatformFeePence }`
     - Call `createInvoice({ workspaceSlug: ws.slug, lineItems, billingPeriodStart, billingPeriodEnd, renewalDate: ws.billingRenewalDate, autoGenerated: true })`
     - Increment created counter
   - Return `{ created, skipped }`
   - Helper `formatPeriod(date: Date): string` -> DD/MM/YYYY format for line item descriptions

   `alertUnpaidBeforeRenewal(): Promise<number>`:
   - Find all invoices where:
     - `status` in ["sent", "overdue"]
     - Invoice's associated workspace has `billingRenewalDate` within the next 48 hours
   - For each: Check workspace's billingRenewalDate. If renewal is within 48h AND invoice is still unpaid:
     - Send Slack alert to admin via `ADMIN_SLACK_CHANNEL_ID` (not workspace channel): "Invoice {invoiceNumber} for {workspaceName} is unpaid — renewal in {hours}h"
     - Use `notify()` from `@/lib/notify` for in-app + Slack: type "system", severity "warning"
   - Return count of alerts sent

2. **Create `src/lib/invoices/overdue.ts`:**

   `markAndNotifyOverdueInvoices(): Promise<number>`:
   - Find all invoices where `status = "sent"` AND `dueDate < now()`
   - For each:
     - Update status to "overdue" via `prisma.invoice.update`
     - **Reminder email** (only if `reminderSentAt` is null — prevents duplicate reminders):
       - Fetch workspace for billing email
       - If workspace has `billingClientEmail`, send reminder email via `sendOverdueReminderEmail()` (defined in this file or notifications.ts)
       - Set `reminderSentAt` to now
     - **Slack alert** to admin via `notify()`: type "system", severity "error", title "Invoice overdue: {invoiceNumber}", message "{workspaceName} — £{total} due {dueDate}"
   - Return count of overdue invoices processed

   `sendOverdueReminderEmail(invoice: InvoiceWithLineItems, recipientEmail: string): Promise<void>`:
   - Use `sendNotificationEmail` from `@/lib/resend`
   - Subject: "Reminder: Invoice {invoiceNumber} is overdue"
   - HTML: Branded email (same style as other project emails — OUTSIGNAL header, body with invoice details, CTA "View Invoice" linking to PDF via viewToken)

#### Task 2: Merge invoice checks into existing daily cron

**Type:** auto

**Files:**
- `src/app/api/inbox-health/check/route.ts`

**Action:**

Add invoice generation, overdue detection, and 48h alert to the existing GET handler in `/api/inbox-health/check/route.ts`.

1. Add imports at the top:
   ```typescript
   import { generateDueInvoices, alertUnpaidBeforeRenewal } from "@/lib/invoices/generator";
   import { markAndNotifyOverdueInvoices } from "@/lib/invoices/overdue";
   ```

2. Inside the `try` block, AFTER the session refresh section (after `refreshStaleSessions()` call), add:
   ```typescript
   // --- Invoice Auto-Generation ---
   const invoiceGenResult = await generateDueInvoices();
   console.log(
     `[${timestamp}] Invoice generation: ${invoiceGenResult.created} created, ${invoiceGenResult.skipped} skipped`
   );

   // --- Overdue Invoice Detection ---
   const overdueCount = await markAndNotifyOverdueInvoices();
   if (overdueCount > 0) {
     console.log(`[${timestamp}] Overdue invoices: ${overdueCount} marked overdue`);
   }

   // --- 48h Unpaid Renewal Alert ---
   const unpaidAlertCount = await alertUnpaidBeforeRenewal();
   if (unpaidAlertCount > 0) {
     console.log(`[${timestamp}] Unpaid renewal alerts: ${unpaidAlertCount} sent`);
   }
   ```

3. Add invoice stats to the return JSON object:
   ```typescript
   invoicesGenerated: invoiceGenResult.created,
   invoicesSkipped: invoiceGenResult.skipped,
   overdueInvoices: overdueCount,
   unpaidRenewalAlerts: unpaidAlertCount,
   ```
