        const iframe = e.currentTarget;
        if (iframe.contentDocument?.body) {

    If `htmlBody` is null, render `bodyText` as `<pre className="whitespace-pre-wrap">` or plain paragraph
  - Show intent/sentiment badges if present (small colored badges: interested=green, positive=green, negative=red, neutral=gray)

  - Find the most recent inbound message that has `aiSuggestedReply !== null`
  - If found, render `<AISuggestionCard suggestion={msg.aiSuggestedReply} onUse={handleUseSuggestion} />`

```typescript
useEffect(() => {
  const fetchThreads = async () => {
    try {
      const res = await fetch('/api/portal/inbox/email/threads');
      if (!res.ok) throw new Error('Failed to load');
      const data = await res.json();
      setThreads(data.threads);
      setError(null);
    } catch (err) {
      setError('Failed to load inbox');
    } finally {
      setLoading(false);
    }
  };

  fetchThreads(); // initial load

  const getInterval = () =>
    document.visibilityState === 'visible' ? 15_000 : 60_000;

  let timer = setInterval(fetchThreads, getInterval());

  const handleVisibility = () => {
    clearInterval(timer);
    timer = setInterval(fetchThreads, getInterval());
  };
  document.addEventListener('visibilitychange', handleVisibility);
  return () => {
    clearInterval(timer);
    document.removeEventListener('visibilitychange', handleVisibility);
  };
}, []);
```

When a thread is selected, render `<EmailThreadView threadId={selectedThreadId} onReplySent={fetchThreads} />` in the right panel.

When no thread is selected, show a placeholder in the right panel: centered mail icon + "Select a conversation" text.

Loading state: show skeleton cards in the thread list panel.

Page title: "Inbox" — use the same page header pattern as other portal pages.

Auto-select first thread on initial load if threads exist and none is selected.
