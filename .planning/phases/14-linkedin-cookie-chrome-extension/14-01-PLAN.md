Define `ExtensionSession` interface:
```typescript
interface ExtensionSession {
  role: "extension";
  workspaceSlug: string;
  senderId: string;
  exp: number; // unix timestamp (seconds)
}
```

Implement:
- `signExtensionToken(session: ExtensionSession): string` — base64url encode JSON payload, HMAC-SHA256 sign with `EXTENSION_TOKEN_SECRET` env var, return `payload.signature` format.
- `verifyExtensionToken(token: string): ExtensionSession | null` — split on `.`, verify HMAC signature, check expiry, return parsed session or null.
- `createExtensionToken(workspaceSlug: string, senderId: string): string` — convenience wrapper that creates session with 7-day expiry and signs it.
- `getExtensionSession(request: NextRequest): ExtensionSession | null` — extract Bearer token from Authorization header, verify and return session.

Use a NEW env var `EXTENSION_TOKEN_SECRET` (not the admin one). Use `createHmac` and `timingSafeEqual` from Node.js `crypto` module for signing/verification, matching the admin-auth pattern.

Export the `ExtensionSession` interface and all 4 functions.

**4. POST /api/extension/senders/[id]/cookies** (`src/app/api/extension/senders/[id]/cookies/route.ts`)
