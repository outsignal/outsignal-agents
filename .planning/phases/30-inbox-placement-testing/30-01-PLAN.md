   - getTestAddress(apiKey: string): calls mail-tester.com API to generate a unique test email address. The API is at `https://www.mail-tester.com/api?key={apiKey}&format=json` — this returns a test address like `test-xxx@srv1.mail-tester.com`. Return { testAddress: string, testId: string }.
   - fetchTestResults(testId: string, apiKey: string): calls `https://www.mail-tester.com/api?key={apiKey}&id={testId}&format=json` to get results. Returns parsed MailTesterResponse or null if not ready yet.
   - pollForResults(testId: string, apiKey: string, maxAttempts = 6, intervalMs = 10000): polls fetchTestResults every 10s up to 60s. Returns MailTesterResponse or null (still pending). Uses setTimeout-based delay loop (not setInterval). Respects Vercel 60s timeout — if polling exhausts attempts, returns null gracefully.

   - isRecommendedForTesting(senderEmail: string): returns boolean — convenience check for a single sender.
