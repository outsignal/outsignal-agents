    - Empty text -> null

    checkGreeting(text, isFirstStep):
    - isFirstStep=false -> always null (no check on follow-ups)
    - isFirstStep=true, text starts with "Hi {FIRSTNAME}," -> null
    - isFirstStep=true, text starts with "Hello {FIRSTNAME}," -> null
    - isFirstStep=true, text starts with "Hey {FIRSTNAME}," -> null
    - isFirstStep=true, no greeting -> hard violation
    - Empty text + isFirstStep=true -> hard violation

    checkCTAFormat(text):
    - Text ending with "?" -> check banned CTAs
    - "worth a chat?" in last sentences -> hard violation (banned CTA)
    - "open to exploring?" -> hard violation (banned CTA)
    - "thoughts?" -> hard violation (banned CTA)
    - "interested?" -> hard violation (banned CTA)
    - "make sense?" -> hard violation (banned CTA)
    - "ring any bells?" -> hard violation (banned CTA)
    - "sound familiar?" -> hard violation (banned CTA)
    - "make sense for your team?" -> hard violation (banned CTA)
    - Text ending without "?" -> hard violation (statement CTA)
    - "open to a quick call this week?" -> null (good CTA)
    - "want me to send over some examples?" -> null (good CTA)
    - Empty text -> null

    checkLinkedInSpintax(text):
    - Text with {option1|option2} -> hard violation
    - Text with {FIRSTNAME} (no pipe) -> null (variable, not spintax)
    - Text with no braces -> null
    - Empty text -> null

    checkSubjectLine(text):
    - Subject with "!" -> hard violation
    - 7-word subject -> soft violation (over 6 word limit)
    - 3-word subject -> null
    - 6-word subject -> null
    - Empty text -> null

    - Word count: `text.trim().split(/\s+/).filter(Boolean).length`
    - 10% grace: `softLimit = Math.floor(limit * 1.1)`. Over limit but under softLimit = soft. Over softLimit = hard.
    - Greeting regex: `/^(Hi|Hello|Hey)\s+\{[A-Z]+\}/i` on trimmed text
    - Spintax regex: `/\{[^{}|]+\|[^{}]+\}/` (pipe inside braces = spintax)
    - CTA check scans last 2 sentences: `text.trim().split(/(?<=[.!?])\s+/).slice(-2)`
    - Subject line word count: same split method, limit 6 words, no grace period for exclamation marks
    - "free" as banned: use word boundary `/\bfree\b/i` to avoid matching "freestyle", "freedom"
