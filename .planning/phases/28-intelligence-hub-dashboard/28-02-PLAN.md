1. **campaign-summary.tsx** — Export `CampaignSummary`:
   - Props: `{ campaigns: CampaignData[] | null; loading: boolean }` (import CampaignData from analytics component).
   - Render top 5 campaigns (`.slice(0, 5)`) in a compact table: rank, name (truncated to ~25 chars), reply rate, interested rate.
   - Show Skeleton when loading. Empty state: "No campaign data yet."
   - "View details" links to `/analytics?tab=performance`.

2. **classification-donuts.tsx** — Export `ClassificationDonuts`:
   - Props: `{ intentData: { intent: string; count: number }[] | null; sentimentData: { sentiment: string; count: number }[] | null; loading: boolean }`.
   - Render two side-by-side Recharts PieChart donuts (innerRadius=35, outerRadius=55, height=140).
   - Intent donut colors: interested=#22c55e, meeting_booked=#10b981, objection=#ef4444, referral=#3b82f6, not_now=#f59e0b, unsubscribe=#dc2626, out_of_office=#6b7280, auto_reply=#9ca3af, not_relevant=#d4d4d8.
   - Sentiment donut colors: positive=#22c55e, neutral=#6b7280, negative=#ef4444.
   - Labels below each donut: "Intent" and "Sentiment".
   - Show Skeleton when loading. Empty state if both arrays empty.
   - "View details" links to `/replies`.

3. **benchmarks-summary.tsx** — Export `BenchmarksSummary`:
   - Props: `{ data: { workspace: string; metrics: Record<string, number>; globalAvg: Record<string, number>; industry: any } | null; loading: boolean }`.
   - Render 3 mini ReferenceGauge components (reuse the existing `ReferenceGauge` from `@/components/analytics/reference-band-gauge`) for replyRate, openRate, interestedRate.
   - If no workspace selected (global), show a message: "Select a workspace to see benchmarks" since benchmarks require a specific workspace.
   - Show Skeleton when loading.
   - "View details" links to `/analytics?tab=benchmarks`.
   - Note: Benchmarks are always all-time data — do NOT pass period param.

4. **icp-summary.tsx** — Export `IcpSummary`:
   - Props: `{ buckets: { bucket: string; totalSent: number; replyRate: number; interestedRate: number }[] | null; recommendation: { current: number; suggested: number; confidence: string; reason: string } | null; loading: boolean }`.
   - Render a compact Recharts BarChart (height=140) showing ICP score buckets on x-axis, replyRate bars.
   - Below chart: if recommendation exists, show a card with the ICP threshold recommendation: "Adjust threshold from {current} to {suggested}" with confidence badge and reason text.
   - Show Skeleton when loading. Empty state: "Not enough data for ICP calibration."
   - "View details" links to `/analytics?tab=benchmarks`.

5. **insights-summary.tsx** — Export `InsightsSummary`:
   - Props: `{ insights: InsightData[] | null; loading: boolean; onRefresh: () => void }` (where InsightData matches the shape from insight-card.tsx).
   - Show count badge: "{N} active insights".
   - Render top 2-3 insights (sorted by priority, `.slice(0, 3)`) using the existing `InsightCard` component from `@/components/analytics/insight-card`.
   - Pass `onAction` that calls PATCH `/api/insights` with the action, then calls `onRefresh` to refetch.
   - Show Skeleton when loading. Empty state: "No active insights. Check back after the next weekly analysis."
   - "View details" links to `/analytics?tab=insights`.
